Deep Linear Networks: Hessian Residual Term

The Hessian of the mean-squared-error loss for a deep linear network decomposes as $H = H_{\text{GN}} + R$, where $H_{\text{GN}} = J^\top J$ is the Gauss–Newton term and $R$ is a residual term that vanishes at the optimum (where $f(x_a) - y_a = 0$). The widget on the main page predicts the sharpness purely from $H_{\text{GN}}$, whose nonzero eigenvalues are $s_i(t) + s_k(t)$. Away from the optimum the residual reshapes the spectrum — and, crucially, lifts a large block of GN-null directions off zero, producing eigenvalues (including negative ones) that $H_{\text{GN}}$ cannot see.

For a 2-layer linear network of input dim $d$, hidden width $m$, and output dim $n$, learning a rank-$r$ target, the full-Hessian eigenvalues split into the following classes. We write $s_i = s_i(t)$ for the current singular value and $s_i^0$ for its target.

  • Aligned ($i = k \le r$): GN eigenvalue $2 s_i$, full eigenvalue $3 s_i - s_i^0$. Count $r$. These carry the residual error and hold the largest eigenvalues.
  • Aligned (null): the GN-null partner of each aligned mode, full eigenvalue $s_i^0 - s_i$. Count $r$. At initialization this sits near $+s_i^0$ — a source of the high early sharpness.
  • Cross ($i \ne k$, both $\le r$): for each unordered pair $\{i,k\}$ the full Hessian mixes a 4-dimensional subspace $\{h_{ik}, h_{ki}, g_{ik}, g_{ki}\}$ into two $2\times2$ blocks, giving the four roots $$\lambda_{1,2} = \tfrac12\!\left[\,2(s_i+s_k) - (s_i^0+s_k^0) \pm \sqrt{4 s_i s_k + (s_i^0-s_k^0)^2}\,\right],$$ $$\lambda_{3,4} = \tfrac12\!\left[\,(s_i^0+s_k^0) \pm \sqrt{\bigl(2(s_i-s_k)-(s_i^0-s_k^0)\bigr)^2 + 4 s_i s_k}\,\right].$$
  • Single-value (one of $i,k > r$): full eigenvalue equals the lone singular value $s_i$ (or $s_k$) — the residual acts trivially here, so this matches GN exactly. Count $r(n-r) + r(d-r)$.
  • Hidden null (±): when the hidden layer is overparameterized ($m > r$), the residual lifts the non-idle GN-null subspace into pairs $\pm(s_i - s_i^0)$, each branch with multiplicity $(m-r)$, for a total count $2r(m-r)$. The GN eigenvalue is $0$ for all of these, so they are invisible to the Gauss–Newton prediction, yet at initialization they sit near $\pm s_i^0$ and dominate the early sharpness. This class is the main correction the residual adds.
  • Hidden null (0): the remaining surplus hidden units pair with idle (zero-singular-value) input/output directions and stay at exactly $0$ for all $t$. Count $(m-r)(n+d-2r)$. These carry no dynamics, but they complete the spectrum — the six classes together account for all $P = m(n+d)$ parameters — so the overlay can plot them as a flat zero line for direct comparison against a full numerical diagonalization.

Turn on full Hessian below to overlay the refined prediction (dash-dot), and switch to top-k per class to display each class independently — including the two hidden-null families as separate curves. At convergence the residual vanishes and the full prediction collapses back onto the Gauss–Newton curves; during training they differ in informative ways — most visibly the high initial sharpness carried by the aligned-null and hidden-null modes.
