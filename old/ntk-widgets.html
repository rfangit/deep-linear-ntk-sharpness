A visualization of the Neural Tangent Kernel for a random model on the dataset $\{x = (1, 0),\ (0, 1)\}$. Note that the target function $y = Mx$ does not affect the NTK, only the residuals (which will evolve according to the NTK).

Each cell is $\Theta(x, x')_{ij}$ — the inner product of derivatives of $f$ (with respect to different output dimensions) across all parameters. Cells colored by $\Theta_{ij}$ relative to the matrix max.

Time evolution of the NTK on the Saxe analytic trajectory. Specify the target singular values $s_1, s_2$ and a basis seed for $U, V$. The model's singular values evolve from $\sigma_0 = 0.001$ according to $\sigma_i(t) = s_i / \big(1 + (s_i/\sigma_0 - 1) e^{-2 s_i t}\big)$. Scrub $t$ to watch the NTK at successive snapshots — its entries change in magnitude, but the eigenbasis is fixed.

Each cell is $\Theta(x, x')_{ij}$ at the current $t$, computed from the analytic snapshot model via Jacobians. Cells colored by $\Theta_{ij}$ relative to the matrix max.

Eigenstructure of the NTK along the same Saxe trajectory. We diagonalize $\Theta(t)$ numerically (no closed-form formula) and project the residual $\vec r(t)$ onto each eigenvector. The eigenbasis is fixed in $t$ — only the eigenvalues evolve. As you scrub, watch how each $\lambda_\alpha$ rises while $|\vec r \cdot e_\alpha|$ falls — and notice which residual components stay pinned at zero throughout.

Eigenvectors are sorted by $\lambda_\alpha$ at $t = t_{\max}$ and held fixed thereafter — bar position $\alpha$ refers to the same eigenvector across the entire scrub. Same color across the two panels indicates the same eigenvector; hover any bar to highlight its partner.
